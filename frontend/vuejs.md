
# ディレクティブ

|----------------|------------------------------------------------------|
| v-for          | 繰り返し                                             |
| v-bind         | ???                                                  |
| v-show         | 評価式内がtrueの時、その要素をdisplayする            |

| v-on           | イベントハンドリングする                             |
| v-model        | 双方向バインディングする                             |
