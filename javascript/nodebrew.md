
# 概要
node.jsのバージョン切り替えツール

# コマンド
| やること                               | コマンド                          |
|----------------------------------------|-----------------------------------|
| インストールできるバージョンを表示     | nodebrew ls-all                   |
| インストールされているバージョンを表示 | nodebrew ls                       |
| インストール（バイナリ）               | nodebrew install-binary <version> |
| 使うバージョンを指定する               | nodebrew use <version>            |
